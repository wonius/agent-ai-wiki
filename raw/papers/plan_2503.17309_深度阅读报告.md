## 2. 一句话概括

提出LLM+MAP框架，将LLM与PDDL符号规划器结合，实现双手机器人的高效任务分解与协调规划。

---

## 3. 科学问题

### 核心问题
如何实现双手机器人在长程任务中的空间和时间协调规划？

### 背景与重要性
- 双手机器人操作具有很高灵活性，但双手时空协调复杂
- 现有工作主要关注获取人类级操作技能，忽视长程任务规划
- LLM在长程推理和复杂任务中存在幻觉问题

### 现有方法不足
- LLM单独规划缺乏逻辑正确性保证
- LLM+P等方法未成功应用于双手机器人
- 需要有效的任务分解和任务分配

---

## 4. 技术路线

### 核心技术方案：LLM+MAP框架
```
┌─────────────────────────────────────────────────────────────────┐
│                      LLM+MAP 流程                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌────────────────────────────────────────────────────────────┐   │
│   │  1. Spatial Scene Understanding (空间场景理解)             │   │
│   │     • 不可协调区域：单臂可达                            │   │
│   │     • 可协调区域：双臂均可达（重叠区域）                  │   │
│   │     • 基于OWLv2等目标检测确定区域                       │   │
│   └────────────────────────────────────────────────────────────┘   │
│                            │                                        │
│                            ▼                                        │
│   ┌────────────────────────────────────────────────────────────┐   │
│   │  2. PDDL Representation (PDDL表示)                      │   │
│   │     • 定义双手机器人领域                                │   │
│   │     • 转换任务配置为PDDL                             │   │
│   └────────────────────────────────────────────────────────────┘   │
│                            │                                        │
│                            ▼                                        │
│   ┌───────────────────────────────────────────────────────���────┐   │
│   │  3. Partial-Order Plan Generation                       │   │
│   │     • 经典规划器生成部分序规划                         │   │
│   │     • 并行/异步/同步执行                             │   │
│   │     • 逻辑正确性保证                                │   │
│   └────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

### 关键创新

**1. 空间区域定义**
- 不可协调区域：仅单臂可达，左右独立，可并行操作
- 可协调区域：双臂均可达，需协调同步/异步执行

**2. PDDL域定义**
- 基于NICOL半人形机器人
- 遵循LABOR agent控制模式

**3. 部分序规划**
- 双臂协同执行序列
- 逻辑正确性保证

### 实验配置
- 后端：GPT-4o
- 对比：GPT-4o, DeepSeek-V3, o1, R1
- 评估指标：规划时间、成功率、group debits、规划步数减少率

---

## 5. 核心发现

### 主要结论
1. LLM+MAP在**规划时间、成功率**上显著优于直接LLM规划
2. 规划步数减少率高
3. 有效处理双臂协调的空间和时间约束

### 关键数据/指标
- 规划时间显著降低
- 成功率提升
- group debits减少
- 规划步骤减少

### 消融实验
- 验证双臂vs单臂规划差异
- 验证并行vs串行执行

## 6. 创新点
