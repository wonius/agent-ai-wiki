# AgentFold: Long-Horizon Web Agents with Proactive Context Management

**Source**: [arXiv 2510.24699](https://arxiv.org/abs/2510.24699) · 2025-10-10

## Key takeaways

- Proactive context management for long-horizon web agents via proactive context folding
- [[AgentFold]] 30B parameter model surpasses 671B models through context folding strategy
- Novel Fold-Generator module learns to decide when and how to fold context
- Addresses context saturation problem in multi-step web navigation tasks

## Core claims

AgentFold introduces a proactive context management framework where a dedicated Fold-Generator learns to compress dialogue history into structured "folds" before context window saturation occurs, enabling a 30B model to outperform models 20x its size on long-horizon web agent tasks.
