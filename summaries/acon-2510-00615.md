# ACON: Optimizing Context Compression for Long-horizon LLM Agents

**Source**: [arXiv 2510.00615](https://arxiv.org/abs/2510.00615) · 2025-12-01

## Key takeaways

- Adaptive context optimization for long-horizon LLM agents
- Compression criterion optimization + distillation approach
- Achieves 26–54% peak token reduction with maintained or improved task accuracy
- Qwen3-14B shows +20–46% accuracy improvement on evaluation benchmarks (e.g., AppWorld, 8-objective QA)

## Core claims

ACON proposes an adaptive context compression framework for LLM agents, optimizing what information to retain during long task executions. By learning which context tokens are redundant or harmful to agent performance, ACON achieves significant compression while actually improving accuracy — proving compression is itself an effective small-model enhancement strategy.
