# ZeroLeak: Using LLMs for Scalable and Cost Effective Side-Channel Patching

**Source**: [arXiv 2308.13062](https://arxiv.org/abs/2308.13062) · 2023-08-24

## Key takeaways

- GPT-4 can fix 97% of side-channel leakages in microbenchmark, at only ~$1.34 for 33 leaks
- Outperforms Google PaLM (56%) and Meta LLaMA (35%) in patch success rate
- Up to 10× faster than compiler-based Spectre mitigation (lfence)
- First comprehensive study of LLM-based patching for microarchitectural vulnerabilities

## Core claims

This paper presents the first systematic study using LLMs to automatically patch security-critical software against timing attacks and Spectre v1 gadgets. The approach uses zero-shot learning with carefully crafted prompts, combined with dynamic leakage detection tools (Microwalk, Pitchfork, Spectector).

## Notable quotes
