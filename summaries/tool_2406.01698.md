# Demystifying AI Platform Design for Distributed Inference of Next-Generation LLM models

**Source**: [arXiv 2406.01698](https://arxiv.org/abs/2406.01698) · 2024-06-03

## Key takeaways

- GenZ tool: analytical tool for navigating LLM model architectures and AI platform design parameters
- Supports Dense, GQA, MoE, Mamba architectures
- Validated against real hardware with max geomean error of 5.82%
- Identifies compute, memory capacity, memory bandwidth, network latency, and bandwidth requirements

## Core claims

This paper presents GenZ, an analytical tool to efficiently navigate the relationship between diverse LLM model architectures (Dense, GQA, MoE, Mamba), LLM serving optimizations (Chunking, Speculative decoding, quantization), and AI platform design parameters. It estimates LLM inference performance metrics and helps inform hardware accelerator design.

## Notable quotes

> "The hardware requirements to meet Service Level Objectives (SLOs) remain an open research question."
