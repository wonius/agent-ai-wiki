# MuMath-Code: Combining Tool-Use LLMs with Multi-perspective Data Augmentation

**Source**: [arXiv 2405.07551](https://arxiv.org/abs/2405.07551) · 2024-05-13

## Key takeaways

- First method combining tool-use with multi-perspective data augmentation
- Two-stage training: Stage-1 on pure CoT data, Stage-2 on code-nested data
- MuMath-Code-7B: 83.8% on GSM8K, 52.4% on MATH
- MuMath-Code-70B: 90.7% on GSM8K, 55.1% on MATH (SOTA among open methods)

## Core claims

This paper bridges two research trajectories: tool-free data augmentation and tool-use LLMs. It augments math questions via rephrasing, alteration, FOBAR, and BF-Trans methods, then synthesizes code-nested solutions. The two-stage training strategy first enhances pure language reasoning, then equips code generation capability.

## Notable quotes

> "MuMath-Code leverages the advantages of both the external tool and data augmentation."
