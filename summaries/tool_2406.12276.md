# Beyond Tool-Use to Using Real-World Codebases with LLM Agents

**Source**: [arXiv 2406.12276](https://arxiv.org/abs/2406.12276) · 2024-06-19

## Key takeaways

- CodeNav: LLM agent that navigates and leverages previously unseen code repositories
- Automatically indexes and searches over code blocks, finds relevant snippets, imports and executes them
- Unlike tool-use agents requiring manual tool registration, CodeNav works with any codebase
- Compares code-use vs tool-use performance on three benchmarks

## Core claims

This paper addresses the gap between tool-use LLM agents (which require pre-registered tools with manual descriptions) and real-world code navigation. CodeNav automatically indexes target codebases, retrieves relevant code snippets, and uses execution feedback to iteratively generate solutions.

## Notable quotes

> "In contrast to tool-use LLM agents that require 'registration' of all relevant tools via manual descriptions within the LLM context, CodeNav automatically indexes and searches over code blocks in the target codebase."
