# AgentGen: Enhancing Planning Abilities for Large Language Model based Agent via Environment and Task Generation

**Source**: [arXiv 2408.00764](https://arxiv.org/abs/2408.00764) · 2024-08-01

## Key takeaways

- First framework for automatically generating diverse environments and planning tasks
- Uses inspiration corpus for environmental diversity (592 diverse environments)
- Bi-Evol method for bidirectional task difficulty evolution
- AgentGen-tuned Llama-3.1-8B surpasses GPT-3.5; 70B model achieves SOTA

## Core claims

Existing agent training relies on **manually designed** environments and tasks, which is labor-intensive. AgentGen introduces:

1. **Environment Generation**: Use diverse text segments (inspiration corpus) as context for LLM to generate environment specifications in PDDL or other languages

2. **Task Generation with Bi-Evol**: Bidirectional evolution — evolve tasks in both easier and harder directions to create smooth difficulty curves

The framework generates 7246 high-quality trajectories for instruction-tuning, achieving >10x improvement over raw Llama-3.1-8B on in-domain tasks.

## Notable quotes

> AgentGen greatly improves LLMs' planning ability, e.g., the AgentGen instruction-tuned Llama-3.1-8B surpasses GPT-3.5 in overall performance.
