# In-Context Exemplars as Clues to Retrieving from Large Associative Memory

**Source**: [arXiv 2311.03498](https://arxiv.org/abs/2311.03498) · 2023-11-06

## Key takeaways

- Novel perspective: ICL as contextual retrieval from associative memory (Hopfield Networks)
- Theoretical framework connecting ICL to memory retrieval mechanisms
- Proposes more efficient active exemplar selection methods
- Sheds light on how in-context exemplars influence ICL performance

## Core claims

This paper conceptualizes in-context learning (ICL) as contextual retrieval from a model of associative memory, establishing a theoretical framework based on Hopfield Networks. The performance of ICL greatly depends on the exemplars used, yet how to choose exemplars remains unclear due to lack of understanding of how in-context learning works. The authors propose that understanding ICL through the lens of memory retrieval provides new insights into exemplar selection.
