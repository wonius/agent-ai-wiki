# Do Large Language Models Advocate for Inferentialism?

**Source**: [arXiv 2412.14501](https://arxiv.org/abs/2412.14501) · 2024-12-19

## Key takeaways

- Proposes Robert Brandom's inferential semantics as alternative foundational framework for understanding LLMs
- Examines how inferentialism's anti-representationalist stance aligns with Transformer architecture
- Analyzes through ISA (Inference, Substitution, Anaphora) approach — demonstrates LLMs exhibit anti-representationalist properties
- Develops consensus theory of truth for LLMs grounded in interactive normativity via RLHF
- LLMs may challenge traditional assumptions: strict compositionality and semantic externalism

## Core claims

The paper argues that LLMs generate meaning without reference to external world representations, aligning with inferential semantics rather than truth-conditional representationalism. The ISA approach shows LLMs process language in a fundamentally anti-representationalist manner.

## Notable quotes

> LLMs exhibit fundamentally anti-representationalist properties in their processing of language.
