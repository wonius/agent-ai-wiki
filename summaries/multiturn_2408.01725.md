# The Drama Machine: Simulating Character Development with LLM Agents

**Source**: [arXiv 2408.01725](https://arxiv.org/abs/2408.01725) · 2024-09-04

## Key takeaways

- "Drama Machine" 框架：使用多个 LLM agent 模拟角色发展
- 引入 Ego/Superego 双代理机制，实现主观对话与内心独白并行发展
- 应用于两种戏剧场景：面试场景和侦探故事
- 探索 AI 主体性的表现力和内部冲突的模拟

## Core claims

The Drama Machine 协调多个 LLM agent 扮演不同的心理角色（Ego 和 Superego），在角色扮演模拟中实现主体间对话与主体内独白并行发展。Superego 代理作为"超我"，在设定的对话轮次介入，对 Ego 的所言所闻进行反思和批评。该框架有两大目标：实践层面 - 实现更灵活丰富的 AI 角色，能在对话交互中转变视角；理论层面 - 促进社会学、精神分析与计算科学的跨学科交流。
