# AutoDroid: LLM-powered Task Automation in Android

**Source**: [arXiv 2308.15272](https://arxiv.org/abs/2308.15272) · ACM MobiCom '24

## Key takeaways

- First LLM-powered mobile task automation system handling arbitrary Android tasks without manual effort
- Achieves 90.9% action accuracy and 71.3% task completion success rate with GPT-4
- Outperforms GPT-4-powered baselines by 36.4-39.7% in task completion
- Reduces LLM query costs by 51.7% through multi-granularity query optimization

## Core claims

AutoDroid proposes combining commonsense knowledge of LLMs with app-specific domain knowledge through automated dynamic analysis. The system includes: (1) functionality-aware UI representation bridging GUI with LLM, (2) exploration-based memory injection augmenting app knowledge, and (3) multi-granularity query optimization reducing inference costs.
