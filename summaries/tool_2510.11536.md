# CodeWatcher: IDE Telemetry Data Extraction Tool for Understanding Coding Interactions with LLMs

**Source**: [arXiv 2510.11536](https://arxiv.org/abs/2510.11536) · 2025-10-13

## Key takeaways

- Lightweight client-server system for capturing IDE interaction events
- Captures CGT (Code Generation Tool) interactions: insertions, deletions, copy-paste, focus shifts
- Enables post-hoc reconstruction of coding sessions
- Validated with 150 trials, precision up to 1.00, recall up to 1.00

## Core claims

CodeWatcher addresses the lack of detailed telemetry data for understanding how developers interact with AI code generation tools (like GitHub Copilot). The system uses a VS Code plugin, Python RESTful API, and MongoDB backend to log semantically meaningful events without disrupting user workflows.

## Notable quotes

> "CodeWatcher enables both real-time intervention and post-hoc reconstruction of development sessions involving CGTs."
