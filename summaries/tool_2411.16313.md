# CATP-LLM: Empowering Large Language Models for Cost-Aware Tool Planning

**Source**: [arXiv 2411.16313](https://arxiv.org/abs/2411.16313) · 2024-11-25

## Key takeaways

- First framework for cost-aware tool planning with LLMs
- CATP-LLM (Llama2-7B) outperforms GPT-4 in QoP metric by 1.5%-93.9%
- Tool Planning Language (TPL) enables non-sequential planning
- OpenCATP dataset: 11,100 samples including 2,400 for non-sequential planning

## Core claims

This paper addresses a critical gap in LLM tool planning: overlooking execution costs. The proposed CATP-LLM framework introduces a Tool Planning Language (TPL) for efficient concurrent tool execution, and a cost-aware offline reinforcement learning algorithm (CAORL) to optimize performance-cost trade-off.

## Notable quotes

> "CATP-LLM with the small Llama2-7B as backbone significantly outperforms GPT-powered methods in terms of higher QoP."

> "Prior studies overlook the tool execution costs, leading to the generation of expensive plans whose costs outweigh their benefits."
