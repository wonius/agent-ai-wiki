# Powering Job Search at Scale: LLM-Enhanced Query Understanding in Job Matching Systems

**Source**: [arXiv 2509.09690](https://arxiv.org/abs/2509.09690) · 2025-08-19

## Key takeaways

- Unified LLM-powered query understanding framework for job search
- Jointly models user query and contextual signals (profile attributes)
- Generates structured interpretations driving personalized recommendations
- Reduces system complexity and operational overhead
- Shows improved relevance in online A/B testing

## Core claims

Traditional job search relies on multiple task-specific NER models (brittle, expensive). This paper introduces a unified framework where LLM jointly models query + context signals to generate structured interpretations, improving scalability and adaptability.

## Notable quotes

> Traditional approaches often rely on multiple task-specific Named Entity Recognition models to extract structured facets as seen in job search applications.
