# Understanding the Planning of LLM Agents: A Survey

**Source**: [arXiv 2402.02716](https://arxiv.org/abs/2402.02716) · 2024-02-05

## Key takeaways

- First systematic taxonomy of LLM-based agent planning
- Five categories: Task Decomposition, Plan Selection, External Module, Reflection, Memory
- Comprehensive analysis of each direction with representative works
- Future challenges discussed: planning accuracy, efficiency, generalization

## Core claims

This survey provides the **first systematic view** of LLM-based agent planning. The proposed taxonomy categorizes existing works into five mainstream directions:

1. **Task Decomposition**: Divide complex tasks into sub-tasks (CoT, ReAct, HuggingGPT)
2. **Multi-plan Selection**: Generate multiple plans, select optimal (ToT, GoT, CoT-SC)
3. **External Module**: Use external planners (LLM+P, LLM+PDDL)
4. **Reflection & Refinement**: Reflect on failures, refine plans (Reflexion, CRITIC)
5. **Memory-augmented**: Leverage memory for planning (REMEMBER, MemoryBank)

## Notable quotes

> Planning, as one of the most critical capabilities for agents, requires complicated understanding, reasoning, and decision-making progress.
