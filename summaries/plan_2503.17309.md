# LLM+MAP: Bimanual Robot Task Planning using Large Language Models and Planning Domain Definition Language

**Source**: [arXiv 2503.17309](https://arxiv.org/abs/2503.17309) · 2025-03-21

## Key takeaways

- First LLM + symbolic planning framework for bimanual robots
- Integrates LLM reasoning with multi-agent planning (PDDL)
- Generates partial-order plans with logical correctness guarantee
- Outperforms direct LLM planning (GPT-4o, o1, R1) in success rate and efficiency

## Core claims

Bimanual robotic manipulation requires **spatial and temporal coordination** between two hands — existing works focus on manipulation skills but neglect high-level task planning.

LLM+MAP addresses:
1. **LLM weaknesses**: long-horizon reasoning errors, hallucinations, no logical correctness guarantee
2. **Bimanual challenges**: task decomposition + efficient task allocation between two arms

The framework:
- Uses LLM to transform task descriptions into PDDL representation
- Generates **partial-order plans** through classical symbolic planners
- Enables efficient parallel execution for left/right hands

## Notable quotes

> We demonstrate superior performance of LLM+MAP, while also providing insights into robotic reasoning.
