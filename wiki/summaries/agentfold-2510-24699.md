# AgentFold: Long-Horizon Web Agents with Proactive Context Management

**Source**: [RuiYe et al., Tongyi Lab, Alibaba — arXiv 2510.24699](https://arxiv.org/abs/2510.24699) · 2025-10-29

## Key takeaways

- **Proactive Context Folding** 机制：Agent 每步可选择粒度凝缩（保留关键结论）或深度整合（抽象掉已完成子任务），100步后上下文仅 ~7k tokens
- 30B 模型（AgentFold-30B-A3B）在 BrowseComp/WideSearch/GAIA 上超越 671B DeepSeek-V3.1 和 o4-mini
- 关键信息存活率：100步 **92%** vs ReAct 的 36.6%，500步仍可工作
- 纯 SFT 足够，无需 RL；折叠决策内化到模型能力中

## Core claims

AgentFold 针对 LLM Web Agent 在长程任务中的上下文饱和根本矛盾，发明了双尺度主动上下文折叠机制。架构核心是"动态认知工作空间"——由多尺度状态摘要序列（s(x_i, x_j)）+ 最新交互（Latest Interaction）组成。每步四元响应：Thinking + Folding Directive + Explanation + Tool Call。Folding Directive 决定是将上步压缩为粒度凝缩（s(x_{t-1})）还是与先前摘要深度整合为抽象结论。训练通过 Fold-Generator 自动生成带折叠动作的轨迹数据，无需人工标注。

## Notable quotes
