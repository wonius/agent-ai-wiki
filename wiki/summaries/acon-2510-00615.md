# ACON: Agent Context Optimization via Learnable Compression Criteria

**Source**: [Minki Kang et al., KAIST/Microsoft Research — arXiv 2510.00615](https://arxiv.org/abs/2510.00615) · 2025-01-02

## Key takeaways

- ACON 核心发现：压缩准则（compression criterion）Prompt 本身是可优化资产，在自然语言空间做梯度更新，无需更新模型参数
- 对比轨迹（contrastive trajectories）提供密集反馈信号：完整上下文成功 vs 压缩上下文失败
- 26-54% peak tokens 降低，小模型 Agent 精度提升 20-46%
- 与 KV Cache 压缩正交，可叠加；支持多模态 Agent

## Core claims

ACON 解决的核心问题是小模型 Agent 的上下文压缩困惑：压缩太多丢失关键信息，压缩不够上下文溢出。核心洞察是压缩准则本身可以被优化——通过 GRPO 在自然语言空间直接优化压缩 Prompt，而非压缩具体的 KV Cache。压缩层次是"语义级"，保留异构上下文信息（因果/状态/前置条件），与 KV Cache 压缩正交。
