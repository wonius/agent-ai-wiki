# Agent 学习框架

**创建时间**: 2026-04-11

## 概述

本学习框架聚焦 Agent 方向的 6 个核心技术领域，涵盖：
- Plan: 任务规划与分解
- Context: 上下文管理
- Tool Match: 工具匹配
- Query Analysis: 查询分析与拆解
- Multi-Turn: 多轮对话
- Anaphora: 指代消解

## 目录结构

- [Plan 规划](learning/01-plan.md) - 任务规划与分解
- [Context 上下文管理](learning/02-context.md) - 对话历史与上下文管理
- [Tool Match 工具匹配](learning/03-tool-match.md) - 用户意图到工具的映射
- [Query 分析与拆解](learning/04-query-analysis.md) - 用户查询的理解与结构化
- [Multi-Turn 多轮对话](learning/05-multi-turn.md) - 多轮对话状态管理
- [Anaphora 指代消解](learning/06-anaphora.md) - 代词/指示代词消解

## 使用方法

1. 阅读各主题框架，了解核心概念
2. 关联论文，填充技术细节
3. 实践中验证和迭代
